# Adamas CGPA/SGPA Calculator

A web application to calculate CGPA and SGPA for Adamas University students.

## Features

- SGPA Calculator
- CGPA Calculator
- CGPA Prediction
- GPA Target Calculator
- Percentage Calculator
- Semester History tracking

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd Adamas-CGPA-SGPA

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Build

```sh
npm run build
```
